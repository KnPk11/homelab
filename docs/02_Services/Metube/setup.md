# Metube Setup

> [!NOTE]
> **Tags:** #Metube #VideoDownloader #YouTube #Media #DockerCompose

## 1. Description

Metube is a self-hosted web GUI for `yt-dlp`, providing a simple interface to download videos from YouTube and other supported sites directly to your server.

## 2. Installation

Add the Docker Compose stack to Portainer and start it.

## 3. Usage

1. Access the web interface at `http://[HOST-IP]:8084`.
2. Paste the URL of the video or playlist you wish to download.
3. Downloads will be saved to the mapped `/mnt/pool/Downloads/Metube` directory.
