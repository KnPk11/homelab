# Docker Bench Setup

> [!NOTE]
> **Tags:** #DockerBench #Security #Docker

## 1. Installation
Clone the repository:

   ```bash
   git clone https://github.com/docker/docker-bench-security.git
   ```

## 2. Usage
Run the audit:

   ```bash
   cd docker-bench-security
   sh docker-bench-security.sh
   ```
